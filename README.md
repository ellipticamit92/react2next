# React2Next ⚡

Convert React apps into Next.js automatically.

React2Next is an open-source CLI tool that helps developers migrate existing React applications to modern Next.js architecture.

## Features

- Convert React Router to Next.js routing
- Generate Next.js project structure
- Convert <img> to next/image
- Detect pages automatically
- AI assisted migration (coming soon)

## Installation

npx react2next migrate ./my-react-app

## Example

Before:

src/pages/About.js

After:

app/about/page.tsx
